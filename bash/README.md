# Bash - Раздел GigaDevOps Base

Задания по написанию Bash скриптов для автоматизации и администрирования.

---

## 📁 Структура

### [zadanie_1](./zadanie_1/) - Задание 1: Скрипт мониторинга системы и управления логами

**Требования:**
- Мониторинг CPU, памяти, диска
- Разделение STDOUT и STDERR
- Ротация логов (3 дня)
- Обработка сигналов (SIGINT, SIGTERM)
- Режим отладки (set -x)

**Статус:** 🟡 В работе

---

### [zadanie_2](./zadanie_2/) - Задание 2: Система резервного копирования и проверки целостности

**Требования:**
- Инкрементальное копирование (tar/rsync)
- Проверка целостности (md5sum)
- Логирование результатов
- Настройка через cron

**Статус:** 🟡 В работе

---

### [zadanie_3](./zadanie_3/) - Задание 3: Анализ процессов и диагностика системы

**Требования:**
- Анализ ресурсов (top, htop, /proc)
- Создание процессов-зомби и сирот
- Анализ системных вызовов (strace)
- Изменение приоритетов (nice, ionice)

**Статус:** 🟡 В работе

---

### [zadanie_4](./zadanie_4/) - Задание 4: Автоматизация управления пользователями

**Требования:**
- Создание пользователей из CSV
- Генерация SSH ключей
- Управление группами и правами
- Удаление пользователей

**Статус:** 🟡 В работе
