## Основные инструменты

- **DNS:** dig, nslookup, BIND9
- **HTTP:** curl
- **SSL/TLS:** OpenSSL
- **Веб-сервер:** Nginx

---

## Ключевые концепции

### DNS
- A-записи (IPv4 адреса)
- CNAME (алиасы)
- MX (почтовые серверы)
- NS (серверы имён)
- PTR (обратное разрешение)

### HTTP/HTTPS
- Методы: GET, POST, PUT, DELETE
- Статус-коды: 200 OK, 201 Created, 204 No Content
- TLS/SSL шифрование

### HTTP/2 vs HTTP/1.1
- Мультиплексирование
- Сжатие заголовков
- Бинарный протокол
- Server Push
