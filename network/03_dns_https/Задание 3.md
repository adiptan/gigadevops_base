### Пункт 2: GET-запрос к публичному сайту

**Команда:**
```bash
curl -i https://jsonplaceholder.typicode.com/posts/1
```

**Вывод:**
```
HTTP/2 200 
date: Sun, 08 Mar 2026 05:34:52 GMT
content-type: application/json; charset=utf-8
content-length: 292
server: cloudflare
x-ratelimit-limit: 1000
x-ratelimit-remaining: 999

{
  "userId": 1,
  "id": 1,
  "title": "sunt aut facere repellat provident occaecati excepturi optio reprehenderit",
  "body": "quia et suscipit\nsuscipit recusandae consequuntur expedita et cum\nreprehenderit molestiae ut ut quas totam\nnostrum rerum est autem sunt rem eveniet architecto"
}
```

**Результат:**
- Статус: **200 OK**
- Протокол: HTTP/2
- Content-Type: application/json
- Получен объект поста с id=1

✅ GET-запрос выполнен успешно

---

### Пункт 3: POST-запрос с тестовыми данными

**Команда:**
```bash
curl -i -X POST https://jsonplaceholder.typicode.com/posts \
  -H "Content-Type: application/json" \
  -d '{"title":"Test Post","body":"This is a test post","userId":1}'
```

**Вывод:**
```
HTTP/2 201 
date: Sun, 08 Mar 2026 05:34:56 GMT
content-type: application/json; charset=utf-8
content-length: 87
location: https://jsonplaceholder.typicode.com/posts/101

{
  "title": "Test Post",
  "body": "This is a test post",
  "userId": 1,
  "id": 101
}
```

**Результат:**
- Статус: **201 Created**
- Location: /posts/101
- Создан новый пост с id=101
- Сервер вернул созданный объект

✅ POST-запрос выполнен успешно (ресурс создан)

---

### Пункт 4: PUT-запрос для обновления данных

**Команда:**
```bash
curl -i -X PUT https://jsonplaceholder.typicode.com/posts/1 \
  -H "Content-Type: application/json" \
  -d '{"id":1,"title":"Updated Title","body":"Updated body content","userId":1}'
```

**Вывод:**
```
HTTP/2 200 
date: Sun, 08 Mar 2026 05:35:00 GMT
content-type: application/json; charset=utf-8
content-length: 90

{
  "id": 1,
  "title": "Updated Title",
  "body": "Updated body content",
  "userId": 1
}
```

**Результат:**
- Статус: **200 OK**
- Пост с id=1 обновлён
- Сервер вернул обновлённый объект

✅ PUT-запрос выполнен успешно (ресурс обновлён)

---

### Пункт 5: DELETE-запрос для удаления данных

**Команда:**
```bash
curl -i -X DELETE https://jsonplaceholder.typicode.com/posts/1
```

**Вывод:**
```
HTTP/2 200 
date: Sun, 08 Mar 2026 05:35:04 GMT
content-type: application/json; charset=utf-8
content-length: 2

{}
```

**Результат:**
- Статус: **200 OK**
- Пост удалён
- Возвращён пустой JSON {}

✅ DELETE-запрос выполнен успешно (ресурс удалён)

*Примечание: JSONPlaceholder возвращает 200 вместо 204, но это особенность тестового API.*
