## Технические детали

**Используемые технологии:**
- Nginx (установлен через Homebrew)
- Порт: 8088 (вместо стандартного 80/8080)
- Виртуальный хост: mysite.local
- Корневая директория: /var/www/mysite

**Особенности macOS:**
- Конфигурация Nginx: `/opt/homebrew/etc/nginx/`
- Виртуальные хосты: `/opt/homebrew/etc/nginx/servers/`
- Управление сервисом: `brew services` (вместо systemctl)
