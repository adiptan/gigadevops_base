## Основные команды Vim

### Режимы работы

**Vim имеет несколько режимов:**
- **Normal mode** (обычный) - для навигации и команд (по умолчанию)
- **Insert mode** (вставка) - для ввода текста (клавиша `i`)
- **Visual mode** (визуальный) - для выделения текста (клавиша `v`)
- **Command mode** (командный) - для выполнения команд (клавиша `:`)

**Переход между режимами:**
- `Esc` - выход в Normal mode
- `i` - Insert mode (вставка перед курсором)
- `a` - Insert mode (вставка после курсора)
- `v` - Visual mode
- `:` - Command mode

## Навигация

```
h - влево
j - вниз
k - вверх
l - вправо

w - следующее слово
b - предыдущее слово

0 - начало строки
$ - конец строки

gg - начало файла
G - конец файла

:<номер> - перейти к строке
```

---

## Редактирование

```
i - вставка перед курсором
a - вставка после курсора
o - новая строка ниже
O - новая строка выше

x - удалить символ
dd - удалить строку
dw - удалить слово
d$ - удалить до конца строки

yy - копировать строку
yw - копировать слово
p - вставить после курсора
P - вставить перед курсором

u - отменить последнее действие
Ctrl+r - повторить отмененное действие

. - повторить последнюю команду
```

---

## Поиск и замена

```
/текст - поиск вперёд
?текст - поиск назад
n - следующее совпадение
N - предыдущее совпадение

:%s/старое/новое/g - заменить все вхождения "старое" на "новое"
:%s/старое/новое/gc - замена с подтверждением
:s/старое/новое/ - замена в текущей строке
```

---

## Работа с файлами

```
:w - сохранить
:w файл - сохранить как
:q - выйти
:q! - выйти без сохранения
:wq - сохранить и выйти
:x - сохранить и выйти (альтернатива :wq)

:e файл - открыть файл
:r файл - вставить содержимое файла
```

---

## Работа с окнами

```
:split - разделить окно горизонтально
:vsplit - разделить окно вертикально
Ctrl+w w - переключение между окнами
Ctrl+w q - закрыть текущее окно
:only - оставить только текущее окно
```

---

## Настройки

```
:set number - показать номера строк
:set nonumber - скрыть номера строк
:set relativenumber - относительная нумерация

:set wrap - включить перенос строк
:set nowrap - выключить перенос строк

:set syntax=python - включить подсветку синтаксиса для Python
:syntax on - включить подсветку синтаксиса
:syntax off - выключить подсветку синтаксиса

:set tabstop=4 - размер табуляции
:set expandtab - заменять Tab на пробелы
```

---

## Дополнительные команды

```
:help - открыть справку
:help команда - справка по команде

:! команда - выполнить shell команду
:r! команда - вставить вывод команды

:set list - показать невидимые символы
:set nolist - скрыть невидимые символы

gg=G - автоматическое форматирование всего файла
```
