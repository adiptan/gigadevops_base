### 3. Обновить систему и установить обновления

**Команда:**
```bash
apt-get update -y && apt-get upgrade -y
```

- [ ] Скриншот выполнения команды

---

### 4. Сгенерировать SSH ключ и подключиться к машине по SSH

**Команды:**
```bash
# Генерация ключа
ssh-keygen -t ed25519 -C "your_email@example.com"

# Копирование ключа на удалённый сервер
ssh-copy-id user@host

# Подключение
ssh user@host
```

- [ ] Скриншот генерации ключа
- [ ] Скриншот успешного подключения по SSH

## 📸 Скриншоты

Для выполнения задания необходимо прикрепить скриншоты для каждого пункта:

1. **Установка Ubuntu** - системная информация (`uname -a`, `lsb_release -a`)
2. **Настройка сети** - вывод `ip a` или `/etc/netplan/*.yaml`
3. **Samba** - конфигурация `/etc/samba/smb.conf` и статус `systemctl status smbd`
4. **SSH** - содержимое `~/.ssh/id_ed25519.pub` и успешное подключение
