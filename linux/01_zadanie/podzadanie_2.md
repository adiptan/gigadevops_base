## 📋 Задание

Для каждого этапа предоставить:

1. **Описание:** Что происходит на этом этапе
2. **Команды:** Как проверить/посмотреть информацию
3. **Скриншоты:** Вывод команд или лог-файлы

**Полезные команды:**
```bash
# Просмотр загрузочных сообщений
dmesg | less

# Systemd targets
systemctl list-units --type=target

# Время загрузки
systemd-analyze
systemd-analyze blame

# Лог загрузки
journalctl -b
```

---

## 📸 Скриншоты

Примеры скриншотов для задания:

1. **GRUB меню** - выбор ядра при загрузке
2. **dmesg** - сообщения ядра при загрузке
3. **systemd-analyze** - время загрузки системы
4. **systemctl list-units** - запущенные сервисы
5. **journalctl -b** - лог последней загрузки
6. **Схема процесса загрузки** - нарисованная схема с этапами
