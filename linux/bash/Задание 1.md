
## 5. Обработка сигналов

### Команда:
```bash
./system_monitor.sh &
PID=$!
sleep 2
kill -SIGTERM $PID
```

### Результат:
```
[2026-02-28 12:05:00] INFO: Получен сигнал завершения. Останавливаю мониторинг...
[2026-02-28 12:05:00] INFO: Скрипт завершён корректно
```

## 6. Режим отладки

### Команда:
```bash
DEBUG_MODE=1 ./system_monitor.sh 2>&1 | head -20
```
