## 1. Скрипт: backup_system.sh

**Расположение:** `zadanie_2/backup_system.sh`

### Запуск:
```bash
./backup_system.sh [директория]
```

## 3. Проверка созданного архива

### Команда:
```bash
ls -lh backups/
```

### Результат:
```
-rw-r--r-- 1 root root 1.2K backup_20260228_122653.tar.gz
```

---

## 4. Проверка контрольных сумм

### Команда:
```bash
cat checksums.md5
```

### Результат:
```
b9930f838aa15ba98e793c5b4b8b5f29  backup_20260228_122653.tar.gz
```

### Проверка целостности:
```bash
md5sum -c checksums.md5
```

### Результат:
```
backup_20260228_122653.tar.gz: OK
```

---

## 5. Содержимое архива

### Команда:
```bash
tar -tzf backups/backup_20260228_122653.tar.gz | head -10
```

### Результат:
```
./
./backup_system.sh
./backup.log
./users.csv
```

---

## 6. Настройка cron

### Команда:
```bash
crontab -e
```

### Добавить строку:
```cron
# Ежедневный бэкап в 02:00
0 2 * * * /path/to/backup_system.sh /important/data >> /var/log/backup_cron.log 2>&1
```

### Проверка cron задач:
```bash
crontab -l
```

### Результат:
```
0 2 * * * /path/to/backup_system.sh /important/data >> /var/log/backup_cron.log 2>&1
```
