### Пункт 2.2 - Просмотр /proc/[PID]/fd

**Команда:**
```bash
ls -la /proc/86958/fd/
```

**Вывод:**
```
total 0
dr-x------ 2 root root  3 Mar  5 20:00 .
dr-xr-xr-x 9 root root  0 Mar  5 20:00 ..
lr-x------ 1 root root 64 Mar  5 20:00 0 -> /dev/null
l-wx------ 1 root root 64 Mar  5 20:00 1 -> /dev/null
lrwx------ 1 root root 64 Mar  5 20:00 2 -> socket:[412915]
```

**Анализ файловых дескрипторов:**

| FD | Режим | Назначение | Куда указывает |
|----|-------|------------|----------------|
| 0 | `lr-x` | stdin (read-only) | `/dev/null` |
| 1 | `l-wx` | stdout (write-only) | `/dev/null` |
| 2 | `lrwx` | stderr (read-write) | `socket:[412915]` |

**Символьные ссылки:**
```
0 -> /dev/null
1 -> /dev/null
2 -> socket:[412915]
```

**Объяснение:**
- **FD 0-2** - стандартные потоки (stdin/stdout/stderr)
- **> /dev/null** перенаправил stdout в "чёрную дыру"
- **socket** для stderr - связь с родительским процессом
- **lr-x/l-wx/lrwx** - права доступа (read/write/readwrite)

### Пункт 3.3 - Сопоставление данных

**Таблица сравнения:**

| Параметр | /proc/meminfo | free -h | Совпадение |
|----------|---------------|---------|------------|
| **Total** | 8131360 kB = 7.76 GB | 7.8 Gi | ✅ (округление) |
| **Free** | 1021048 kB = 997 MB | 997 Mi | ✅ Точное совпадение |
| **Available** | 6487240 kB = 6.19 GB | 6.2 Gi | ✅ (округление) |
| **Buffers** | 224948 kB | - | В buff/cache |
| **Cached** | 5399176 kB = 5.15 GB | - | В buff/cache |
| **buff/cache** | Buffers + Cached = 5.4 GB | 5.5 Gi | ✅ Сумма |
| **Swap Total** | 0 kB | 0 B | ✅ Swap отключён |

**Формулы free -h:**
- `total` = `MemTotal` из /proc/meminfo
- `used` = `total - free - buffers - cache`
- `free` = `MemFree`
- `shared` = `Shmem`
- `buff/cache` = `Buffers + Cached`
- `available` = `MemAvailable` (доступная для приложений)

**Объяснение параметров:**

| Параметр | Описание |
|----------|----------|
| **MemTotal** | Общий объём физической памяти |
| **MemFree** | Полностью свободная память |
| **MemAvailable** | Доступная для новых приложений (включая освобождаемые кэши) |
| **Buffers** | Память для блочных устройств |
| **Cached** | Page cache (кэш файлов) |
| **Active** | Недавно использованная память |
| **Inactive** | Редко используемая память (кандидат на освобождение) |
| **Dirty** | Данные для записи на диск |
| **Mlocked** | Заблокированная память (не может быть вытеснена) |

**Вывод:** 
- `/proc/meminfo` предоставляет детальную информацию в килобайтах
- `free -h` показывает агрегированные данные в человекочитаемом формате
- Значения полностью совпадают (с учётом округления)
