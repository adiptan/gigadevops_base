## Пункт 1.1 - Определение процессов в системе


**Найденные shell-процессы:**

- PID 84854: bash (pts/1) - активный shell

## Пункт 2 - Дерево процессов

**Команда:**
```bash
pstree -p | head -30
```

**Вывод:**
```
systemd(1)-+-ModemManager(948)-+-{ModemManager}(956)
           |                   |-{ModemManager}(961)

           |                  |-{containerd}(1082)
           |                  |-{containerd}(1367)
           |                  `-{containerd}(34358)

```

**Анализ:**
- **Родительский процесс всех:** `systemd (PID 1)` - система инициализации
- **Примеры дочерних процессов:**
  - `NetworkManager (889)` → 3 потока {NetworkManager}
  - `containerd (1053)` → 8 потоков для управления контейнерами
  - `ModemManager (948)` → 3 потока

**Формат дерева:**
- `-+-` указывает на ветвление (parent → children)
- `{}` обозначают потоки (threads) процесса
- Числа в скобках - PID процессов
