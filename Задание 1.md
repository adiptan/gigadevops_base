# Задание 1: Создание и разметка дисков

## Требования
1. Создать 3 виртуальных диска объёмом по 1ГБ
2. Разметить диски:
   - disk1.img в формате MBR, один раздел ext4
   - disk2.img в формате GPT, два раздела: xfs и btrfs
   - disk3.img используя parted для гибкой разметки
3. Настройка файловой системы и проверка состояния

```bash
partprobe /dev/loop4
```

## Пункт 2.2: Разметка disk2.img в GPT, два раздела (xfs и btrfs)

### Установка необходимых утилит:
```bash
apt-get install -y xfsprogs btrfs-progs
```

### Команды:
```bash
# Создание GPT таблицы
parted /dev/loop16 mklabel gpt

# Создание первого раздела для xfs (512 МБ)
parted /dev/loop16 mkpart primary xfs 1MiB 512MiB

# Создание второго раздела для btrfs (остальное место)
parted /dev/loop16 mkpart primary btrfs 512MiB 100%

# Обновление таблицы разделов
partprobe /dev/loop16

# Проверка разметки
parted /dev/loop16 print
```

### Вывод:
